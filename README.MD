### Installation


In the project folder, run this ...

```sh
$ npm install
$ cd seed
$ node product-seeder.js
$ cd ..
$ npm start
```

Verify the deployment by navigating to your server address in your preferred browser.

```sh
127.0.0.1:3000
```
